# Katalon Studio plugin for TestRail integration

## setup
Need below two private libraries available 
``` xml
<dependency>
    <groupId>io.railflow</groupId>
    <artifactId>commons</artifactId>
    <version>1.0</version>
</dependency>
<dependency>
    <groupId>io.railflow</groupId>
    <artifactId>testrail-client</artifactId>
    <version>2.1</version>
</dependency>
```
